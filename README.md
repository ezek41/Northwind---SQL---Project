### Project Overview

* **Database Engine:** SQL Server (SSMS)
* **Language:** T-SQL
* **Visualization:** Google Sheets, Excel
* **Workflow:** Set up the database, then run the queries.

---

### Query Roadmap

* **Query 1 (EDA):** Quick overview of total customers, orders, and core metrics.
* **Query 2:** Top 10 customers by billing/purchase volume.
* **Query 3:** Total billing per customer (without the top 10 filter).
* **Query 4:** RFM Model (Recency, Frequency, Monetary value).
* **Query 5:** EDA and data cleaning for order and shipping dates.
* **Query 6:** Excluding orders missing a shipping date (these require a separate analysis beyond our current scope).
* **Query 7:** Processing days between purchase and shipment per customer (filtered for top accounts).
* **Query 8:** Processing days between purchase and shipment broken down by country.
